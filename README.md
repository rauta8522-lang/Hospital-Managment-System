# Hospital-Managment-System

"A simple and efficient Hospital Management System built using C for the backend logic and HTML for the user interface. This project is designed to manage patient records, appointments, and hospital data seamlessly."

🚀 Features
Patient Management: Add, update, and view patient details.

Appointment Scheduling: Manage doctor-patient consultations.

Data Persistence: Uses C for robust data handling.

User-Friendly UI: Clean interface built with HTML.

🛠️ Tech Stack
Language: C

Frontend: HTML

Platform: GitHub Pages (for hosting the UI)

📁 File Structure
hospital_system.c: Contains the core logic and data management functions.

index.html: The web interface for interacting with the system.

README.md: Project documentation.

⚙️ How to Run
For the UI: Simply open index.html in any web browser.

For the Backend:

Compile the C file: gcc hospital_system.c -o hospital_system

Run the executable: ./hospital_system
